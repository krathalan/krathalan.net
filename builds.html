builds.krathalan.net

You can find packages for Arch Linux here. You may either browse and download the packages and signatures manually, or add the repo to your /etc/pacman.conf.

Browse

x86_64 packages: https://builds.krathalan.net/x86_64/
aarch64 (ARMv8) packages: https://builds.krathalan.net/aarch64/

Add to pacman

Add the following snippet to your /etc/pacman.conf:
[krathalan]
Server = https://builds.krathalan.net/$arch

You will need to add my GPG key to your pacman keyring and trust it:
$ curl -O https://builds.krathalan.net/keys/srht.asc
$ sudo -s
# pacman-key -a srht.asc
# pacman-key --edit-key B46B326273E4A1D21AAA3F6F529AC10050BD24EF

Then trust the key, save, and exit. Now you can pacman -Syu and get some packages.
