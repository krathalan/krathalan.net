You can find packages I have built for for Arch Linux here. You may either browse and download the packages and signatures manually, or add the repo to your /etc/pacman.conf.

Modifications

Three available packages are built with modifications to their PKGBUILD and/or other files, or different-than-default configuration options (click to see patches for that package):

  • linux-mainline: no kexec, only supports AMD/Intel CPUs, stronger ASLR (like linux-hardened)[↗], no SMACK/SELinux/Tomoyo support, GCC plugins (entropy+randstruct)
  • ungoogled-chromium: enable ozone flags
  • wine-tkg-git: proton patches, -march=znver2[↗]

All other packages are built with their default state after fetching from the AUR.

Add to pacman

Add the following snippet to your /etc/pacman.conf:

[krathalan]
Server = https://builds.krathalan.net/$arch

You will need to add my GPG key to your pacman keyring and trust it:

$ curl -O https://builds.krathalan.net/keys/srht.asc
$ sudo -s
# pacman-key -a srht.asc
# pacman-key --edit-key B46B326273E4A1D21AAA3F6F529AC10050BD24EF

Then trust the key, save, and exit. Now you can pacman -Syu and get some packages.
